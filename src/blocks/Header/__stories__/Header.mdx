## Parameters

There could be only one one Header block on the page.

`type: 'header-block'`

`title: text | TitleProps` ‚Äî Title. Can be a string or an object with `text`, `textSize`, `url`, and other [TitleProps](#titleprops) fields. Defaults to `'l'` text size.

`switchingTitle?: {text: string; switchingTime?: number}` ‚Äî Title that switches some words in accordance w/ the syntax.

`description?: text` ‚Äî Subtitle.

`width?: 's' | 'm' | 'l'` ‚Äî Title text width.

`buttons?: button[]` ‚Äî List of Button blocks.

`offset?: 'default' | 'large'` ‚Äî Top and bottom margins.

`image?: string | {src: string; alt?: string;}` ‚Äî Image to the right of the text.

`stock?: 'price' | 'shares'` ‚Äî Stock type.

`stockPrice?: { name?: string; percent?: string; price?: string; update?: string; delayed?: string; }` ‚Äî Stock price data.

`stockShares?: { date?: string; total?: string; text?: string; included?: { title: string; text: string; }[]; description?: string; }` ‚Äî Stock shares data.

`video?:`

-   `src: string[]`
-   `loop?:`
    -   `start: number`
    -   `end: number`

`backLink?: 'url' | 'title'`

`imageSize?: 's' | 'm'` ‚Äî Image size.

`background?:`

-   `fullWidth: true | false` ‚Äî Background width: Color to the entire screen width or as per column size.
-   `fullWidthMedia: true | false` ‚Äî Media width: Picture or Video to the entire screen width or as per column size.
-   `color: string` ‚Äî Background color
-   `image?: string | ImageObjectProps | ImageDeviceProps`
-   `video?: Video`

`theme?: default | dark` ‚Äî Sets the text color (black by default, white for the dark background).

`unicorn?: string` ‚Äî Link to the embed json file.

`unicornSdkUrl?: string` ‚Äî Link to the unicorn sdk.

`unicornFallbackImage?: string` ‚Äî Static decorative image shown when the Unicorn scene is disabled for reduced motion, constrained device/network signals, runtime jank, or QA fallback testing.

`unicornFallbackImageMobile?: string` ‚Äî Mobile static decorative image for the same fallback path.

`forceMobileImage?: boolean` ‚Äî Show the mobile fallback image immediately instead of the Unicorn scene on mobile devices. Defaults to `true`; set to `false` to keep adaptive Unicorn behavior on mobile.

`verticalOffset?: '0' | 's' | 'm' | 'l' | 'xl'` ‚Äî Top and bottom offsets from the text. (Values: '0' - 0px, 's' - 48px, 'm' - 80px, 'l' - 112px, 'xl' - 144px.)

`verticalOffsetTop?: '0' | 's' | 'm' | 'l' | 'xl'` ‚Äî Top offset override verticalOffset

`verticalOffsetBottom?: '0' | 's' | 'm' | 'l' | 'xl'` ‚Äî Bottom offset override verticalOffset

`backgroundEffect?:`

-   `firstSrc: string` ‚Äî url video without hover
-   `secondSrc: string` ‚Äî url video with hover

`headerSpace?: boolean` ‚Äî Take into account the height of the header menu

</StoryTemplate>

`mediaView?: full | fit` ‚Äî full is default value, the media content covers all width and height. Fit - media content has real sizes and locates in the middle
