## Parameters

`type: "form-block"`

`title?: string` - Title.

`textContent?` - See the [Content](?path=/story/components-content--docs&viewMode=docs) component.

`textFormContent?` - See the [Content](?path=/story/components-content--docs&viewMode=docs) component.

`formData?: {yandex?: Object, hubspot?: Object}` - See [Yandex](?path=/story/components-yandexform--docs&viewMode=docs) / [Hubspot](?path=/story/components-hubspotform--docs&viewMode=docs) form data.

`direction?: 'form-content' | 'content-form' | 'center'` - Direction.

`fullWidth?: boolean` - if direction equal 'center' and fullWidth equal true then form use 100% width of block.

`fullWidthContent?: boolean` - if direction equal 'center' and fullWidthContent equal true then content use 100% width of block.

`background?: BackgroundImage` — See [background](?path=/story/components-pics-video-datalens-backgroundimage--docs&viewMode=docs) properties.

`image?: string` — Alternative option for setting the background of an image

`backgroundColor?: string` — Alternative option for setting background color

`slug?: string` — To use the form from the context.
