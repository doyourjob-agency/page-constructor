## Parameters

`type: 'logo-rotator-block'`

`animated?: bool` — Enables/disables animation for the block (enabled by default).

`title?: Title | string` — Title

`items: {url?: string; src: string; isStatic?: boolean}[]` — Items for slots.

`count: number` — Rendering amount of items for mobile and as the default when `desktopCount` is not set.

`desktopCount?: number` — Rendering amount of items for desktop viewports (`md` and wider).

`minRotateCount?: number` — Minimum amount of non-static visible logos rotated per batch.

`maxRotateCount?: number` — Maximum amount of non-static visible logos rotated per batch.

`swapAnimation?: 'fade' | 'morph'` — Logo swap animation. Defaults to `fade`, which fades the old logo out before fading the new logo in; `morph` enables the layered morph-like transition.

Every 2s the block picks a random batch of non-static visible logos between `minRotateCount` and `maxRotateCount`. Hovering a logo pauses only that logo, and selected slots switch using `swapAnimation`.

`colSizes?: Object` — Sizes of item, more info [here](?path=/docs/documentation-types--docs#colsizes).

`rowMode?: boolean` — Rendering in row.
